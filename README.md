# Python Port Scanner 🔍

## Descrizione
Tool sviluppato in Python per la scansione di porte TCP utilizzando multithreading.

## Funzionalità
- Scansione porte 1–1024
- Multithreading per velocizzare il processo
- Rilevamento base dei servizi

## Tecnologie utilizzate
- Python
- Socket
- Threading

## Utilizzo
```bash 
scan python.py```
